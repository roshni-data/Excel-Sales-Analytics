# Excel-Sales-Analytics

## Description
This project focuses on analyzing customer performance using advanced Excel techniques.
It helps in evaluating customer performance and identifying key business insights.

## Tools Used
-Excel
## Process/Key steps
1.Built data model by creating relationships between miltiple tables using power pivot

2.Used pivot tables for summarizing and analyzing data

3.Created calculated columns using RElATED function

4.Developed custom measures using DAX formulas

5.Applied CALCULATE function for dynamic filtering

6.Compared 21 vs 20 net sales using DIVIDE function

7.Applied Conditional Formatting to highlight key insights and trends

8.Improved data readability by converting numbers into Millions(M) 

## Business Insights:
1.Focus more on high-Performing customer

2.Improve strategies for low-performing customer

3.Invest in high-growth segments

4.Optimize sales strategy

## Result:
1.Sales increased from 2020 to 2021

2.Identified top and low-Performing customers

3. year-over-year growth observed
