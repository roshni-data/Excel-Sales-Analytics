# Excel-Sales-Analytics

## Description
This project focuses on analyzing customer performance using advanced Excel techniques.
It helps in evaluating customer performance and identifying key business insights.

## Tools Used
-Excel
## Process/Key steps
1.Built data model by creating relationships between miltiple tables using power pivot

2.Used pivot tables for summarizing and analyzing data

3.Created calculated columns using RElATED function

4.Developed custom measures using DAX formulas

5.Applied CALCULATE function for dynamic filtering

6.Compared 21 vs 20 net sales using DIVIDE function

7.Applied Conditional Formatting to highlight key insights and trends

8.Improved data readability by converting numbers into Millions(M) 

## Business Insights:
1.Focus more on high-Performing customer

2.Improve strategies for low-performing customer

3.Invest in high-growth segments

4.Optimize sales strategy

## Result:
1.Sales increased from 2020 to 2021

2.Identified top and low-Performing customers

3. year-over-year growth observed

## Project Overview
This project analyzes market performance across different countries by comparing actaul sales with predefined target.This report helps identify underperforming regions and evaluate overall business performance

## Objectives
1. Compare actaul sales vs target for each country

2. Identify underperforming and better-performing regions
 
3. Analyze year-wise sales growth(2019-2020)
 
4. Support data-driven business decisions
 
## Used tool
-Excel

## Data source
1. The dataset is taken from an existing Excel report
   
2. Data was reused using Copy & Move method
  
3. Further analysis and transformation were performed on the dataset

 ## Process Followed
   
### Data Preparation
1. Imported data from an existing Excel file
   
### Data Transformation
1. Organized country-wise and year-wise sales data
   
3. Created calculated fields for analysis
### Analysis performed
1.Compared Net Sales (2019,2020,2021)

2.Calculated Target vs Actual variance

3.Calculated Target % (performance gap)

### Visualization
1.Used tables and conditional formatting

2.Highlighted negative performance using color scales

## Key Insights
1.Overall performance is below target (-9.17%), indicating a gap in achieving business goals

2.Major markets like India and USA contribute high sales but still show negative variance

3.Sales have grown from 2019 to 2021, but targets were still not fully achieved

4.Performance varies across regions, suggesting need for region-specific strategies
## Results
1. Total Sales(2021):598.9M USD
2. Total Target Gap:-54.9M USD
3. Overall Target Achievement:-9.17%
